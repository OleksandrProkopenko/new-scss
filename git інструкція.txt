заходимо в папку де знаходиться .git
git add (шлях до файла який я створив)-додавання нового файла до репозиторія
git commit -a -m "короткий опис"-додавання змінених файлів в коміт
git push origin -закидування файлів в репозиторій 
git pull origin -отримання файлів з репозиторія
